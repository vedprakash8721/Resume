Stock Trend Forecasting — LSTM Web App

Python TensorFlow (Keras) LSTM Streamlit yfinance Time‑Series

Built a production-grade machine learning and deep learning system to forecast stock price trends using LSTM neural networks. Leveraged TensorFlow (Keras), NumPy, Pandas, and yfinance for sequence modeling and data preprocessing. Integrated MinMax scaling, EMA (20/50/100/200) analytics, and walk-forward validation for reliable time-series prediction. Deployed an interactive Streamlit web app enabling parameter tuning (sequence length, train split) and real-time price visualization with MAE/RMSE metrics, CSV/PNG downloads, and an intuitive dashboard for financial insights. Tech Stack: Python, TensorFlow (Keras), LSTM, Pandas, NumPy, yfinance, Scikit-learn, Streamlit, Matplotlib, Time-Series Forecasting
